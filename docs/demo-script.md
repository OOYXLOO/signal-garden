# Demo Script

Target length: 45-60 seconds.

## Shot List

1. Splash screen
   - Show the Signal Garden title and `Play today`.
   - Voiceover: "Signal Garden turns a Reddit post into a tiny daily puzzle."

2. First board
   - Open the game board.
   - Show seed, moves, score, and best score.
   - Voiceover: "Each day has a shared seed, so everyone in the thread solves the same board."

3. Rotate tiles
   - Rotate two or three tiles.
   - Show the signal lighting connected tiles.
   - Voiceover: "Players rotate signal tiles to connect the root to the bloom."

4. Hint and score
   - Press `Hint`.
   - Show the move count and score changing.
   - Voiceover: "The game is quick to learn, but the score creates room to optimize."

5. Copy result
   - Press `Copy result`.
   - Show the comment-ready result line.
   - Voiceover: "When players finish, they can paste a result line into the comments and compare routes."

6. Closing
   - Return to the board with the daily seed visible.
   - Voiceover: "The hook is simple: play today's board, compare paths, come back tomorrow."

## Captions

```text
Daily seed: one shared board per thread
Rotate tiles to route the signal
Score rewards clean paths
Copy a comment-ready result
Play, compare, return tomorrow
```

## Notes

- Do not show private Reddit account pages.
- Do not show credentials, API keys, cookies, payment data, or account settings.
- Keep the demo focused on gameplay and the Reddit comment loop.
